# Setup

## Prerequisites

* [io.js 1.6.3](https://iojs.org/dist/v1.6.3/) - Required to avoid to issues with `electron-rebuild`

### Optional

* Electron (global install)

```
$ npm install electron-prebuilt -g
```

## Install

* `npm install`

## Run

### Run via Node.js/io.js

```
$ node index.js
```

### Run as electron app

```
$ electron app/
```
